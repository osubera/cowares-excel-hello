文字コード変換コマンド type_text

初めにお読みください

このソフトにできること

コマンドラインで、日本語などの文字エンコードを変換する。

euc-jp などでエンコードしたテキストファイルを、ウィンドウズ標準に変換して読む。 ウィンドウズのテキストファイルを、 euc-jp などエンコードを指定して保存する。 BOM 無しの utf-8 ファイルを保存する。 VBScript だけで書かれているので、ほとんどのウィンドウズ環境でそのまま動く。 ライセンスがフリーなので、好き勝手にカスタマイズできる。

動作環境

  • Windows 2000 以降すべて。

配布しているファイルについて

  1. ZIP形式で圧縮しています。 Windows 2000 以前では、この形式を解凍できるソフトウェアが別途必要です。
  2. 圧縮ファイルには以下のファイルが含まれます。
    type.vbsエンコードされたファイルを標準出力に表示するツール。
    text.vbs標準入力をエンコードしてファイルに保存するツール。
    xtype.batファイルを読むときのバッチの見本。
    xtext.batファイルを保存するときのバッチの見本。
    etype.bat etext.bateuc-jp 専用にするときのバッチの見本。
    ライセンス.txtこのソフトウェアのライセンスです。
    初めにお読みください.htmlこの解説書です。
    その他解説書の使う画像ファイルなど。

ご使用の手引き

  1. ファイルを読むときは、 CScript //NoLogo type.vbs /e:文字コード名 読みたいファイル名
  2. ファイルに書くときは、 CScript //NoLogo text.vbs /e:文字コード名 保存するファイル名
  3. 文字コードを指定しないときは utf-8 になります。
  4. 指定できる文字コードの名称と動作は、使用中のウィンドウズに依存します。
  5. utf-8 指定の場合のみ、ウィンドウズの標準動作と異なり、世界標準のBOM無し保存をします。 ただし、これは "utf-8" 指定の場合に限り、別名の unicode-2-0-utf-8 などを使えば、 BOM 付き保存になります。
  6. 日本語を使う人に関係しそうな、主な文字コード名
    • utf-8 shift_jis euc-jp iso-2022-jp unicode
  7. とりあえず使ってみる。
    1. コマンドプロンプトを開きます。
    2. CScript //NoLogo type.vbs /e:iso-2022-jp jis.txt
    3. 上のとおり入力し、エンターを押すと、jis.txt が変換されて画面に表示されます。
    4. CScript //NoLogo text.vbs /e:euc-jp < sjis.txt euc.txt
    5. 上のとおり入力し、エンターを押すと、sjis.txt が変換されて euc.txt に保存されます。

ダウンロードしたプログラムが動かないとき

  • 助けを求める前に、セルフチェックで確認しましょう。
  • スクリプトが動かない人のセルフチェッカー
