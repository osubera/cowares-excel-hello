初めにお読みください

このソフトにできること

コマンドラインで、ウェブページをダウンロードする。

個別ページやファイルをピンポイント指定でダウンロードできる他、URLの一覧をテキストファイルで作っておき、一括して取得することもできる。 VBScript だけで書かれているので、ほとんどのウィンドウズ環境でそのまま動く。 ライセンスがフリーなので、好き勝手にカスタマイズできる。

動作環境

  • Windows 2000 以降すべて。
  • Windows 2000 の場合、以下のコンポーネントが無ければ、 マイクロソフトのページからダウンロード して追加してください。
    • MSXML バージョン6以上
    • MDAC バージョン2.5以上

配布しているファイルについて

  1. ZIP形式で圧縮しています。 Windows 2000 以前では、この形式を解凍できるソフトウェアが別途必要です。
  2. 圧縮ファイルには以下のファイルが含まれます。
    httpget.vbs１つのファイルをダウンロードするツール。
    httpgets.vbs複数のファイルを連続ダウンロードするツール。
    httpgetw.vbsマウスで、複数のファイルを連続ダウンロードするツール。
    example.batバッチファイルで自動化するときの見本。
    urls.txtバッチファイルが使用するサンプルのウェブページリスト。
    ライセンス.txtこのソフトウェアのライセンスです。
    初めにお読みください.htmlこの解説書です。
    その他解説書の使う画像ファイルなど。

ご使用の手引き

  1. 単一ファイルの取得は、 CScript httpget.vbs ダウンロードするURL名 保存するファイル名
  2. 複数ファイルの取得は、 CScript httpgets.vbs < ダウンロードするURLとファイル名のリスト
  3. マウスで操作するのは、 httpgetw.vbs のアイコンにファイル名のリストファイルをかぶせる
  4. とりあえず使ってみる。
    1. コマンドプロンプトを開きます。
    2. CScript httpget.vbs http://cowares.nobody.jp/favicon.ico favicon.ico
    3. 上のとおり入力し、エンターを押すと、カレントディレクトリに favicon.ico が保存されるはずです。

ダウンロードしたプログラムが動かないとき

  • ActiveX コンポーネントはオブジェクトを作成できません。
    1. Windows 2000 に必要なコンポーネントをインストールしてください。
    2. それでもだめなら、次の手順を試してください。
    3. httpget.vbs をメモ帳で開く。
    4. 次の個所をみつけ、 
      Sub Main(Url, FileName)
    Dim tp
    Set tp = CreateObject("MSXML2.XMLHTTP")
    'Set tp = CreateObject("MSXML2.XMLHTTP.6.0")
      次のように修正する。 
      Sub Main(Url, FileName)
    Dim tp
    'Set tp = CreateObject("MSXML2.XMLHTTP")
    Set tp = CreateObject("MSXML2.XMLHTTP.6.0")
      これを保存してから試す。
    5. httpgets.vbs httpgetw.vbs も、似た個所があるので同じ修正をする。
  • アクセスが拒否されました。
    1. このプログラムは、インターネットエクスプローラに依存しています。
    2. インターネットエクスプローラのセキュリティ設定が、該当URLへのアクセスを許可していない場合、このようなメッセージが出る場合があります。
  • 助けを求める前に、セルフチェックで確認しましょう。
  • スクリプトが動かない人のセルフチェッカー
