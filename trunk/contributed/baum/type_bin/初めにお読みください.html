バイナリダンプコマンド type_bin

初めにお読みください

このソフトにできること

コマンドラインで、バイナリダンプとダンプテキストからのバイナリ生成を行う。

バイナリファイルを１６進や２進にダンプし、テキストに変換する。 １６進ダンプテキストからバイナリファイルを生成する。 ２進ダンプを使ってゲージツ的な創作活動もできる。 VBScript だけで書かれているので、ほとんどのウィンドウズ環境でそのまま動く。 ライセンスがフリーなので、好き勝手にカスタマイズできる。

動作環境

  • Windows 2000 以降すべて。

配布しているファイルについて

  1. ZIP形式で圧縮しています。 Windows 2000 以前では、この形式を解凍できるソフトウェアが別途必要です。
  2. 圧縮ファイルには以下のファイルが含まれます。
    typebin.vbsバイナリファイルをダンプして標準出力に表示するツール。
    dechex.vbs標準入力を１６進ダンプとして解釈し、バイナリファイルに保存するツール。
    typebin.batバイナリダンプのバッチの見本。
    dechex.batバイナリファイルを保存するときのバッチの見本。
    ge-jitu.bat２進ダンプの文字を指定して、アスキーアート遊びをするバッチの見本。
    ライセンス.txtこのソフトウェアのライセンスです。
    初めにお読みください.htmlこの解説書です。
    その他解説書の使う画像ファイルなど。

ご使用の手引き

  1. とりあえず使ってみる。
    1. コマンドプロンプトを開きます。
    2. CScript //NoLogo typebin.vbs shop.png
    3. 上のとおり入力し、エンターを押すと、 shop.png の先頭128バイトが変換されて画面に表示されます。
    4. CScript //NoLogo typebin.vbs /c:444 shop.png > shop.txt
    5. 上のとおり入力し、エンターを押すと、 shop.png がすべて変換されて shop.txt に保存されます。メモ帳で確認できます。
    6. CScript //NoLogo dechex.vbs < shop.txt another.png
    7. 上のとおり入力し、エンターを押すと、shop.txt が変換されて another.png に保存されます。
  2. バイナリファイルを生成するときは、 CScript //NoLogo text.vbs 保存するファイル名
    1. バイナリ保存は、１６進ダンプ専用です。
    2. 改行や漢字などは無視するので、メモ帳で１６進ダンプを編集し、バイナリ生成することができます。

    4. プログラム自身もテキストなので、たとえばメールでバイナリ添付無しに、バイナリファイルと再生ツールをセットにして、単純なプレーンテキストとして送信できることになります。
  3. バイナリファイルをダンプするときは、 CScript //NoLogo typebin.vbs 読みたいファイル名
    1. バイナリダンプのオプション
      • /x ―― バイト単位１６進ダンプを指示（デフォルト）
      • /d ―― バイト単位１０進ダンプを指示
      • /o ―― ４ビット単位１０進ダンプを指示
      • /b ―― １ビット単位２進ダンプを指示
      • /c:128 ―― ダンプするバイト数（長さ）を指定（デフォルトは128）
      • /w:16 ―― ダンプ出力の改行幅（入力バイト数）を指定（デフォルトは16）
      • /s:0 ―― ダンプを開始するバイト位置（先頭が0）を指定（デフォルトは0）
      • /bit:Low ―― ２進ダンプのビット順（Low または High）を指定（デフォルトはLow）
      • /bit0:0 ―― ２進ダンプで 0 に割り当てる記号を指定（デフォルトは0）
      • /bit1:1 ―― ２進ダンプで 1 に割り当てる記号を指定（デフォルトは1）
    2. バイナリダンプは、複数のファイルをマウスでドロップして使うこともできます。
    3. ２進ダンプを使えば、適当なバイナリファイルから、アスキーアートのパターン生成ができます。
