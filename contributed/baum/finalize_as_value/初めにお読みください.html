動作環境

  • Excel 97以降すべて。
  • Windows 95 以降すべて。
  • Macintosh での動作は未確認。

配布しているファイルについて

  1. ZIP形式で圧縮しています。 Windows 2000 以前では、この形式を解凍できるソフトウェアが別途必要です。
  2. 通常のブックとアドインが含まれますが、動作は同じです。どちらか一方、好みの形式を使ってください。
  3. 圧縮ファイルには以下のファイルが含まれます。
    finalize_as_value.xlsmExcel 2007 形式のソフトウェア本体です。これをエクセルで開けば使えます。
    finalize_as_value_2k.xlsExcel 2000 形式のソフトウェア本体です。これをエクセルで開けば使えます。
    finalize_as_value.xlamExcel 2007 形式のソフトウェア本体です。アドインとして使います。
    finalize_as_value_2k.xlaExcel 2000 形式のソフトウェア本体です。アドインとして使います。
    ライセンス.txtこのソフトウェアのライセンスです。

ご使用の手引き

エクセル2007
エクセル2000
  1. このソフトウェアを開きます。
  2. 計算値を確定したいエクセルブックを開きます。
  3. アドインリボンかツールバーで、操作を行います。
    1. 「値確定」を押すと確定操作を実行します。
    2. 「選択セル」「シート」「ブック」のいずれかを選ぶと、確定操作の対象範囲を指定できます。
    3. 「すべての式」「別シート参照」「外部ブック参照」のいずれかを選ぶと、確定操作の対象を数式の参照先により限定できます。
    4. 「ログ」スイッチがオンの時は、確定操作の記録をメモ帳に出力します。
  4. とりあえず使ってみる。
    1. 新規ブックを開く。
    2. セルA1に =Now() と入れる。
    3. セルA1にカーソルを移動する。
    4. 値確定ボタンを押す。
    5. セルA1の日付が確定値になったはずです。
  5. ログ出力のサンプル

ダウンロードしたマクロが動かないとき

  • 助けを求める前に、セルフチェックで確認しましょう。
  • エクセルマクロが動かない人のセルフチェッカー
