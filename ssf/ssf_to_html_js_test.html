SSF to HTML

Because the SSF is a simple text format, we can convert it into an HTML Table easily using JavaScript. This is an example of a dynamic SSF viewer on a web browser.


This is a smart way to publish your Excel Spreadsheet in your web pages and your blogs. No painting tools are required. No PDF writers are required. No plugins are required on the client browser. Everything required is free.
